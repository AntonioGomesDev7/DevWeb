1. O que é HTML?

HTML significa HyperText Markup Language (Linguagem de Marcação de Hipertexto). Ela estrutura e organiza o conteúdo de uma página. Pense no HTML como o esqueleto de um site.

Estrutura, semântica, formulários e acessibilidade. CSS: Flexbox, Grid, animações e responsividade. JavaScript (Básico): Tipos de dados, funções, DOM, eventos.

HTML - Estrutura e Semântica O HTML (HyperText Markup Language) é a base de qualquer página web. Aqui estão alguns conceitos essenciais:

Tags mais importantes:

html: A raiz do documento. Tudo dentro de uma página HTML deve estar entre html e /html. head: Parte “invisível” da página. Aqui colocamos informações como o título, codificação, links para CSS e mais. body: A parte “visível”, onde todo o conteúdo real aparece (textos, imagens, botões, etc).

Estrutura básica:

As tags semânticas ajudam a dar sentido ao conteúdo. Isso melhora o SEO (otimização de buscas), a acessibilidade e a organização do código.

Exemplos e funções:

header: Cabeçalho. Normalmente contém o logotipo, menu, título da página.

nav: Navegação. Agrupa links para navegar no site.

main: O conteúdo principal da página.

section: Uma seção. Pode conter um grupo de artigos, textos, ou imagens.

article: Um conteúdo independente. Pode ser um post de blog, uma notícia, etc.

footer: Rodapé. Informações finais como direitos autorais, links, contatos.
